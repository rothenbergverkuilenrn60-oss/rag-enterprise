           │   │    │    └─► ToolResult(_ERROR_MARKER, is_error=True,    │
           │   │    │            metadata={latency_ms, error:True})      │
           │   │    │                                                    │
           │   │    ├─ returns []                                        │
           │   │    │    └─► ToolResult(_EMPTY_MARKER, is_error=False,   │

           │   │         └─► ToolResult("- " + "\\n- ".join(facts),     │
