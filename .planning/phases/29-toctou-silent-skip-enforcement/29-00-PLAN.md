<objective>
Close the TOCTOU race window between the bulk-dedupe SELECT and the
`executemany` INSERT in `LongTermMemory.save_facts` by wrapping the
precheck-plus-INSERT critical section in a PostgreSQL transaction holding a
per-`(user_id, tenant_id)` advisory lock.

Purpose: SK-01 silent-skip semantics in Plan 29-01 are only safe once the race
window is closed. Without the lock, two concurrent writers with the same
(user, tenant, fact-text) can each pass the precheck and each INSERT â€” defeating
the dedupe contract. TOC-01 acceptance: exactly 1 surviving row.

Output: production code on `services/memory/memory_service.py::save_facts`
+ a `tests/integration/memory/test_save_facts_toctou.py` concurrent integration
test that proves the race is closed.

Per `29-CONTEXT.md` D-TOC-01 (advisory-lock path locked): no schema migration,
no UNIQUE constraint, no ON CONFLICT â€” advisory lock alone.
</objective>
