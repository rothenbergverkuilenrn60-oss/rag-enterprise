       ```python
       async def get_current_user(
           credentials: HTTPAuthorizationCredentials | None = Depends(_bearer),
       ) -> AuthenticatedUser:
           """FastAPI dependency: extract + verify JWT Bearer token.

           Returns the authenticated user, or raises 401 with a sanitized detail.
           Used by Phase 5 routes (POST /ingest/async, GET /ingest/status/{task_id})
           per ASVS V2 (Authentication) and V4 (Access Control) requirements.

           if user is None:
               raise HTTPException(status_code=401, detail="Invalid or expired token")
           return user
       ```

| T-05-04 | Information disclosure | 401 error detail | mitigate | Static error strings: "Authorization required" and "Invalid or expired token" — never echo token contents; verified by grep for `credentials.credentials` in error paths |

| T-05-06 | Tampering | get_auth_service singleton | mitigate | Singleton pattern reused (not re-instantiated); prevents per-request state pollution; verified via test_get_current_user_uses_singleton_auth_service |
