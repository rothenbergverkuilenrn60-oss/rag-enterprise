<!-- asciinema cast v2 format (https://docs.asciinema.org/manual/asciicast/v2/) -->

Format: JSON-lines text file. First line is a JSON header object:
```json
{"version": 2, "width": 120, "height": 40, "timestamp": 1715299200, "env": {"SHELL": "/bin/bash", "TERM": "xterm-256color"}}
```
Subsequent lines are event tuples:
```json
[0.123456, "o", "event: planner.plan\ndata: {...}\n\n"]
```

Render: `agg docs/demo.cast docs/demo.gif --speed 1.0 --theme monokai`.
