```text
1. Coordinator decomposes multi-dim query into N sub-questions, returns [query] for single-dim.
2. Sub-agents run concurrently (asyncio.gather).
3. Each sub-agent has isolated message history (no cross-contamination).
4. Hard cap MAX_SWARM_AGENTS prevents unbounded fan-out.
5. Partial sub-agent failure does not crash; final response still returned.
6. Synthesis combines all sub-answers (including error markers) into final answer.
7. Audit log records swarm_n, per_agent_turns, per_agent_tool_calls, swarm_latency_ms.
```

       @pytest.mark.unit
       @pytest.mark.asyncio
       async def test_sub_agents_run_concurrently(mock_pipeline, gen_req):
           mock_pipeline._llm.chat = AsyncMock(side_effect=['["q-A","q-B"]', "synth"])
           started_event = asyncio.Event()
           pending = {"n": 0}
           async def slow_turn(**_):
               pending["n"] += 1
               if pending["n"] >= 2:
                   started_event.set()
               await asyncio.wait_for(started_event.wait(), timeout=2.0)
               return _turn(stop_reason="text_only", text="ans")
           mock_pipeline._llm.call_agentic_turn = AsyncMock(side_effect=slow_turn)
           # If sub-agents ran serially, this would deadlock waiting on the Event.
