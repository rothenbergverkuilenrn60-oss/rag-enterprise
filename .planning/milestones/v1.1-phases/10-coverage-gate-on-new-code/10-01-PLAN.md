- `diff-cover.html` HTML report uploaded as a GitHub Actions artifact
- New `make coverage-diff` Makefile target for local pre-PR checks (compares against `origin/master...HEAD` per D-02)
- README.md `## Testing` section updated with a brief diff-coverage subsection
</objective>
