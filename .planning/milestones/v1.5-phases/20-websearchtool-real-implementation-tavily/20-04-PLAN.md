| T-20-16 | T (Tampering) / I (Information disclosure via XSS) | DOM injection via crafted `metadata.source` host containing HTML/JS | mitigate | `hostOf(url)` returns the parsed `host` (or literal `?`); the value passes through `esc()` before HTML concatenation. `esc` already escapes `<>"'&`.
